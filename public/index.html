Request Header Parser Microservice

This is a microservice that returns information about your browser and system.

It can detect your IP address, preferred language, and browser information.

Example Usage:

/api/whoami

Example Output:

{
  "ipaddress": "192.168.1.1",
  "language": "en-US,en;q=0.5",
  "software": "Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/91.0.4472.124 Safari/537.36"
}
